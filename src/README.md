# JP Morgan Quant Research Simulation

## 📌 Overview
This project models and prices a natural gas storage contract using historical price data and time-series analysis.

The system predicts gas prices based on seasonal trends and evaluates the profitability of storing gas over time.

---

## 🧠 Key Concepts
- Time-series modeling (trend + seasonality)
- Commodity storage economics
- Profit optimization under cost constraints

---

## ⚙️ Features
- Predict future natural gas prices
- Evaluate storage contract profitability
- Support multiple injection and withdrawal dates
- Includes storage, injection, and withdrawal costs

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy

---
